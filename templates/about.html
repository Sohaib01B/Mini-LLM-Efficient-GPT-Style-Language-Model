About This Project

Project Overview

This portfolio showcases a custom-built Mini GPT language model trained from scratch to generate creative stories. The model features a transformer architecture optimized for narrative generation.

Technical Details

  • Architecture: Transformer-based GPT model
  • Parameters: ~23 million parameters
  • Features: Multi-head attention, positional encoding, layer normalization
  • Training: Custom dataset of stories and narratives
  • Framework: PyTorch with Flask web interface

Model Architecture

MiniGPT Model:
├── Embedding Layer (512 dim)
├── 8 Transformer Blocks
│   ├── Multi-Head Attention (8 heads)
│   ├── Feed Forward Network (2048 dim)
│   └── Layer Normalization
├── Output Layer
└── Weight Tying
